# 🛗 Lift Management System (C++)

A console-based **Lift Management System** developed in **C++** that simulates the operation of an elevator in a multi-floor building. The project demonstrates object-oriented programming, menu-driven programming, and basic elevator scheduling logic.

# 📖 Overview

The **Lift Management System** is a C++ console application designed to simulate the behavior of a real-world elevator. Users can request floors, move the lift up or down, and monitor its current position. This project demonstrates core programming concepts while providing a simple simulation of elevator operations.

# ⚙️ How to Run

### Clone Repository

```bash
git clone https://github.com/rajeevbaraila2007-lab/lift-management-system.git
```

### Compile

```bash
g++ main.cpp -o lift
```

### Run

**Windows**

```bash
lift.exe
```

**Linux / macOS**

```bash
./lift
```
